## Keywords

## Research Question / Objective
> How can a low-power wide-area (LPWA) IoT platform — specifically using LoRa — be designed and deployed for intelligent, data-driven irrigation in precision agriculture, with RF energy harvesting to power the IoT nodes?

## Methodology
- **Study type:** System design + platform implementation paper
- **Platform:** AREThOU5A IoT — LoRa radio interface for data forwarding
- **LPWAN technology covered:** LoRa / LPWA (explicitly) ✓ — project aims at accelerating LPWA penetration
- **AI/ML technique:** Not the primary focus — IoT platform and connectivity design
- **Agriculture domain:** Precision irrigation / water management

## Key Findings
1. AREThOU5A platform demonstrates viable LoRa-based LPWA IoT architecture for precision irrigation — sensor nodes forward collected data via LoRa interface.
2. RF energy harvesting applied as alternative power source for IoT nodes — extends battery life without infrastructure dependency.
3. LPWA technologies identified as key enablers for agricultural IoT through low-power, long-range connectivity.
4. System architecture covers: IoT sensor node → LoRa radio → gateway → cloud → irrigation control.
5. Platform validated for rational exploitation of water irrigation in field deployments.

## Limitations Acknowledged by Authors
> - AI/ML analytics not integrated — rule-based irrigation control only
> - RF energy harvesting tested in controlled conditions
> - No comparative LPWAN technology analysis (LoRa only)
